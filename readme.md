Me interesso muito por educação e nesse projeto tento contribuir usando uma pipeline de Análise de Dados para tirar insights das pesquisas feitas pelo IDESP, índice responsável por avaliar o desempenho das escolas no Estado de São Paulo.
Estou aprimorando esse projeto conforme aprendo novas habilidades em dados.

Conclusões até agora: 
1. Descobri quais as escolas com as piores médias no período analisado.
2. Mapeei as escolas com grande melhora nas médias.
3. Há uma grande ausência de dados nos "Anos Iniciais" da pesquisa.

Como usar esses insights?
Sugiro que, tendo mapeado as escolas com o maior progresso de um ano para o outro e as escolas com as piores médias, houvesse posterior investigação de quais métodos foram aplicados no primeiro grupo para tentar implanta-los no segundo.
Além disso, sugiro que haja uma investigação do porquê as escolas tem uma ausência tão grande nos dados dos anos iniciais escolares.

O que ainda pretendo fazer com esse projeto:
1. Integrar um banco de dados mais extenso (pelo menos 10 anos de informações) e avaliar se houve um progresso na média geral de avaliação ao longo de vários anos, isso para saber se a tendência das escolas está em uma crescente (ou seja, ações estão sendo efetivas) ou se não.
2. Há um grande volume de dados ausentes, pretendo comparar com anos anteriores para checar se isso também é uma tendência ou apenas um caso isolado.
3. Pretendo combinar essa pesquisa com um DataSet de Municípios para saber se a diferença entre o desempenho escolar é acentuada pela desigualdade regional.

Dados públicos retirados do site: https://dados.educacao.sp.gov.br/dataset/%C3%ADndice-de-desenvolvimento-da-educa%C3%A7%C3%A3o-do-estado-de-s%C3%A3o-paulo-idesp-por-escola
